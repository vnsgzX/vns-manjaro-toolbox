# 🛠️ VNS Manjaro Toolbox

**VNS Manjaro Toolbox** is a lightweight and efficient GUI maintenance utility specifically designed for **Manjaro Linux** users. It provides a simple, click-and-go interface for common system administration tasks, eliminating the need to remember complex terminal commands.

---

## ✨ Features

* **Mirror Update:** Automatically find and rank the fastest mirrors for quicker downloads.
* **Cache Cleanup:** Free up disk space by removing old and uninstalled package files.
* **System Update:** A one-click shortcut to ensure your entire system is up to date.
* **Native Experience:** Built with Python and PyQt6 to ensure a fast, responsive, and modern look.

---

## 🚀 Installation

To install **VNS Manjaro Toolbox** permanently on your system, run the following commands:

```bash
git clone [https://github.com/vnsgzx/vns-manjaro-toolbox.git](https://github.com/vnsgzx/vns-manjaro-toolbox.git)
cd vns-manjaro-toolbox
makepkg -si
