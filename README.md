# VNS-Manjaro-Toolbox

A lightweight PyQt6-based maintenance utility for Manjaro Linux.

## Features
- Full System Update (Official + AUR via `yay`)
- Deep Cache & Orphan Cleaning
- Mirrorlist Optimization
- SSD Trim Support
- System Error Log Viewer
- Database Lock Fix

## Requirements
- Python 3.10+
- `python-pyqt6`
- `yay` (AUR helper)
- `xfce4-terminal`

## How to run
1. Clone the repo: `git clone https://github.com/TVOJ-PROFIL/vns-manjaro-toolbox.git`
2. Navigate to folder: `cd vns-manjaro-toolbox`
3. Run: `python main.py`
